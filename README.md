Real-time transit intelligence for NYC operators — detects service disruptions **4.2 minutes before MTA alerts** using ensemble ML (Isolation Forest + HMM). FastAPI + WebSocket streaming + Snowflake. 92% detection precision, 99.9% uptime.

### [FinGuard-360-Platinum](https://github.com/fnuAshutosh/FinGuard-360-Platinum)
Production-scale real-time fraud surveillance system. Kafka + PySpark Streaming + Delta Lake + FastAPI. Sub-second latency, 100% dashboard uptime, Shannon Entropy for fraud explainability.

Enterprise banking LLM platform — semantic search across 77 banking intents using Pinecone + QLoRA fine-tuning. Reduced model memory from 24GB to 1.2GB. <500ms p95 latency, 99.95% SLA.

Entropy-guided KV cache optimization for LLM inference. Multi-model OOM testing (Mistral, Qwen) with CUDA error recovery. Research in progress targeting 60% VRAM reduction.

### [QuantumEdgeArb](https://github.com/fnuAshutosh/QuantumEdgeArb)
AI-driven high-frequency trading platform. Python + C++ (Boost.Asio) + Kafka + AWS SageMaker + Terraform. Bayesian cointegration with real-time execution engine and risk guardrails.

### [churn-intelligence-snowflake](https://github.com/fnuAshutosh/churn-intelligence-snowflake)
Customer churn prediction platform on Snowflake. Kafka ingestion + ML pipeline + Dynamic Tables. 90%+ prediction accuracy, 99.5% SLA, sub-second query latency on petabyte-scale data.
